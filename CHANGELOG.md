# Changelog

All notable changes to this project will be documented in this file.

## [0.3.3] - 2026-05-02
- 修复系统文本大小不为100%时界面异常的问题
- 修复连发键位过多时，影响底栏和连发配置栏的问题
- 底栏暂时改为只显示连发和连招的数量


## [0.3.2] - 2026-05-02

### Bug Fixed
- 修复没有自动获取应用权限的问题。
- 修复 Win10 下使用了旧的 comctl32 系统组件的问题。

## [0.3.1] - 2026-05-01

### Bug Fixed
- 修复修改按键连发的时间间隔时，主窗口崩溃的问题。

## [0.3.0] - 2026-05-01

### Added
- 按键连发：支持全局连发按键、按键间隔和重复校验。
- 生效规则：支持“全局 + 当前配置”和“仅当前配置”。
- 一键连招 v1：支持职业配置和自定义配置，包含快捷栏按键动作与顺序手搓动作。
- 配置管理：支持职业配置显示/隐藏、自定义配置新增/删除、当前生效配置切换。
- 悬浮窗：支持打开、关闭、拖动记忆和启动时自动打开。
- 托盘：支持主界面打开、悬浮窗开关、当前配置显示和退出。
- 全局启动/停止快捷键：支持组合键注册与触发。
- 开机自启动：支持 Windows Run 注册表项写入与移除。
- 日志：支持可配置日志等级和按会话分文件记录。
- 浏览器预览：支持 `pnpm dev:web` 下的 mock 模式预览。

### Changed
- 默认连发按键调整为 `X`。
- Windows 安装包改为中文界面，默认只生成 NSIS 发行包。
